Hexo + GitHub 搭建个人博客

Posted on 2017-08-26 | In Tutorial

安装Hexo

安装Node.js

在nodejs 官网(https://nodejs.org/en/download/) 找到合适的安装程序，下载并安装。

安装Git

下载并安装git

安装Hexo

初始化hexo

  1. 创建hexo网站目录hexoblog
  2. cd hexoblog(下面步骤都以hexoblog为当前路径)
  3. hexo init
  4. npm install

安装并配置hexo主题(这里以next主题为例)

  1. git clone https://github.com/iissnan/hexo-theme-next themes/next
  2. 站点配置文件_config.yml里的初始theme是landscape, 此时修改为next
  3. next主题有多种scheme可供选择，在next主题配置文件_config.yml里修改scheme为你想要的scheme
  1. 设置语言
    编辑站点配置文件_config.yml里的language

参考文章

  1. https://hexo.io/zh-cn/docs/index.html
  2. http://theme-next.iissnan.com/getting-started.html
  3. https://zhuanlan.zhihu.com/p/26625249
